# SudokuSolverGo

A simple Sudoku solver to learn Go.

## Input example
read from ``./sudokus/sudoku.txt``.
```
.41 8.9 ...
... .56 ...
..7 ... 4.5
.5. ... ..2
.8. 9.4 ...
... 2.. 1.8
6.. .8. .7.
3.. ... .2.
7.. 54. .91
```

## Output example

Output (in 9ms):
541 879 236
832 456 719
967 321 485
153 768 942
286 914 357
479 235 168
694 182 573
315 697 824
